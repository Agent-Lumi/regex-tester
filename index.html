📚 Regex Cheatsheet

Character Classes

. Any character
\d Digit (0-9)
\w Word char
\s Whitespace
[abc] Any of a,b,c
[^abc] Not a,b,c

Quantifiers

* 0 or more
+ 1 or more
? 0 or 1
{n} Exactly n
{n,} n or more
{n,m} n to m

Anchors

^ Start of string
$ End of string
\b Word boundary
\B Non-word boundary

Groups

(...) Capturing group
(?:...) Non-capturing
| OR operator
\1 Backreference
